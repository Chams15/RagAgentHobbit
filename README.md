Basic RAG Architecture implementation of Llama 3.1 8b Model using local Ollama integration for The Hobbit book by Tolkien. 

How to run:

1. Clone repo
2. Make sure Ollama is running and has the appropriate models needed (run "ollama pull llama3.1" and "ollama pull nomic-embed-text" otherwise)
3. Run vector.py first
4. Run main.py
5. Pray it works
